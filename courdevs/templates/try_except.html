Try Except Example:

  1. First we have a function 'div42by'
    def div42by(divideBy):
    return 42/ divideBy

    print (div42by(2))
    print (div42by(12))
    print (div42by(0))

    #the result will be this error:
    #(ZeroDivisionError: division by zero)

  1. So to Solve that add a Try/Except

    def div42by(divideBy):
    try:
    return 42/ divideBy
    except ZeroDivisionError:

    #('Error: You tried to divide by zero')
