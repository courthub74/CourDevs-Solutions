Git-Git


  1. Initialize Git Repository
  2. git config --global user.name "Your Name"
    git config --global user.email "you@youraddress.com"
    git config --global push.default matching
    git config --global alias.co checkoutgit init
    git add .
    git commit -m "message here"

  3. Basic Git Commands For Git Version Control
  4. git init
    git add .
    git commit -m "message here"
    git status
    git log('q' to exit)
    git checkout(license plate- unique commit id)

  5. Push from Git to GitHub
  6. git remote add origin git@github.com: courthub74/(repo name here)
    git push -u origin master
    If Original Add Origin is Rejected try below:
    git remote set-url origin https://github.com/courthub74/(repo name here)

  7. Existing Push
  8. git status
    git add .
    git commit -m "Commit Message"
    git push origin master
    NOTE: if changes are made in your GitHub i.e. ReadMe commits, then perform below:
    git pull origin master

    THEN perform:
    'git push origin master' again
